# 스터디 허브

논문 스터디, 운동, 독서 스터디, 연구 등 팀 스터디·활동을 한곳에서 관리하는 웹앱입니다.  
React + Vite로 구성되어 있습니다.

## 팀원

- 창민석, 이윤지, 송수현, 강태영, 조수민, 신현호

## 기능

- **홈**: 메뉴 카드로 일정 / 스터디 & 활동 / 팀원 진입
- **참여자 선택**: 상단 "나는 ○○○"로 내 기준 보기
- **일정**: 추가·삭제, **참여자 지정** (일정별로 누가 참여하는지 선택)
- **내 일정만 보기**: "나는 ○○○" 선택 후 **내 일정** 탭으로 필터
- **스터디 & 활동**: 논문 스터디, 운동, 독서 스터디, 연구 카드
- **팀원**: 멤버 목록 보기

일정·참여자 선택은 브라우저 로컬 저장(localStorage)에 저장됩니다.

## 사용 방법

```bash
cd study-hub
npm install
npm run dev
```

브라우저에서 표시되는 주소(예: http://localhost:5173)로 접속합니다.

- `npm run build`: 프로덕션 빌드
- `npm run preview`: 빌드 결과 미리보기
